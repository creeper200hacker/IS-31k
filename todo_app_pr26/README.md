# ПР26 — Flask приложение «Список дел»

Приложение умеет:
- показывать список задач;
- добавлять новые задачи;
- удалять отдельные задачи;
- очищать весь список;
- сохранять задачи в tasks.json;
- автоматически добавлять дату создания задачи.

## Запуск

1. Перейти в папку проекта:

cd todo_app_pr26

2. Установить зависимости:

pip install -r requirements.txt

3. Запустить приложение:

python app.py

4. Открыть страницу:

http://127.0.0.1:5000
